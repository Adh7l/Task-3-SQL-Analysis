Main Learnings

WHERE filters rows before aggregation, HAVING works after aggregation.

Joins help combine data from multiple tables.

Subqueries are useful for comparing each row against an aggregate.

Views help save a query and use it again easily.

ARPU = total revenue / number of customers.

SQL is very helpful for structured data and reporting.
